
# Workbench Guide

## Ziel

Parametrischer Controller-Builder in FreeCAD.

## Aktueller Stand

- Workbench registriert
- erster Command vorhanden
- GUI minimal

## Nutzung

1. FreeCAD starten
2. Workbench „Open Controller“
3. Command ausführen

## Ziel-Workflow

1. Projekt erstellen
2. Gehäuse definieren
3. Komponenten platzieren
4. Cutouts generieren
5. Schema exportieren

## Konzepte

### Controller
Gesamtobjekt

### Surface
Bedienoberfläche

### Component
Bauteil (Encoder, Button etc.)

### Cutout
Ausschnitt

### Keepout
Sperrbereich

### library_ref
Verweis auf Bibliothek

## Ziel UI

- Projekt Wizard
- Component Browser
- Layout Grid
- Validation Panel
- Export Panel

# Screenshots

This directory is reserved for release screenshots and should contain at least:

1. tree view with `OCW_Components` and grouped components visible
2. 3D view showing components together with `OCW_PCB`
3. drag interaction in progress
4. mounting geometry with bosses and screws

Recommended file names:

- `01-tree-components.png`
- `02-3d-components-pcb.png`
- `03-drag-interaction.png`
- `04-mounting-bosses-screws.png`

The release documentation and GitHub release should use real captures from FreeCAD, not illustrative mockups.

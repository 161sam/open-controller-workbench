# Examples

These example files are meant to make the first OCW session understandable in a few minutes.

## Demo Projects

- `projects/pad_grid_demo.yaml`
  - pad grid template
  - component groups in the tree
  - visible PCB and mounting structure
- `projects/encoder_module_demo.yaml`
  - encoder-focused controller
  - visible components and grouped controller structure
- `projects/fader_strip_demo.yaml`
  - fader-based controller layout
  - good for drag and spacing workflows
- `projects/full_controller_demo.yaml`
  - broader mixed controller example

## Suggested Demo Order

1. Start with `pad_grid_demo.yaml`
2. Open `encoder_module_demo.yaml`
3. Finish with `fader_strip_demo.yaml`
